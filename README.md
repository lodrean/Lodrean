## 📱 Избранные проекты

### [TestBinlist](https://github.com/lodrean/TestBinlist) · Kotlin
Приложение для определения информации о банковской карте по BIN.
- **Stack:** Jetpack Compose, Ktor Client, Room, Koin
- **Фичи:** валидация BIN по алгоритму Луна, Snackbar-ошибки, кэширование истории
- **Качество:** Unit-тесты (Turbine + MockK), ProGuard/R8, Detekt в CI
- [![CI](https://github.com/lodrean/TestBinlist/actions/workflows/build.yml/badge.svg)](https://github.com/lodrean/TestBinlist/actions/workflows/build.yml)

### [FindYourCourses](https://github.com/lodrean/FindYourCourses) · Kotlin
Клиент-серверное приложение для поиска и записи на курсы.
- **Stack:** Jetpack Compose, Ktor, Koin, Room, Coroutines, Flow
- **Архитектура:** MVVM + Clean Architecture

### [PlaylistMaker](https://github.com/lodrean/PlaylistMaker) · Kotlin
Приложение для создания плейлистов и поиска треков.
- **Stack:** XML layouts, Retrofit, Room, Coroutines, MediaPlayer
- **Фичи:** поиск треков через iTunes API, плейлисты, аудиопревью

### [EffectiveMobileTest](https://github.com/lodrean/EffectiveMobileTest) · Kotlin
Тестовое задание с демонстрацией навыков работы с сетью и UI.
- **Stack:** Jetpack Compose, Retrofit, Coroutines

### [Navigation3](https://github.com/lodrean/Navigation3) · Kotlin
Эксперименты с Jetpack Navigation 3 и новыми паттернами навигации.

### [KtorBaseSettings](https://github.com/lodrean/KtorBaseSettings) · Kotlin
Базовый шаблон бэкенда на Ktor с настройками аутентификации и маршрутизации.

### [DrawingOnCanvasCompose](https://github.com/lodrean/DrawingOnCanvasCompose) · Kotlin
Рисование на Canvas в Jetpack Compose — демо кастомных графических элементов.
